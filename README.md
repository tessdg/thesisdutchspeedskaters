# Gender Differences in Online Criticism of Dutch Speed Skaters
### Bachelor Thesis — Neuro Psychology | Erasmus University Rotterdam

## Project Overview
This project investigates whether negative criticism directed at professional Dutch speed skaters on Reddit differs as a function of the athlete's gender. Specifically, it examines differences in attribution type (performance vs. person-directed criticism) and gender-based criticism using computational methods.

## Research Question
*"To what extent does the nature of negative criticism of professional Dutch speed skaters on Reddit differ as a function of the athlete's gender?"*
